# 🚀 Переїзд бота на хостинг (Railway)

Зараз бот працює тільки коли в тебе відкрите вікно терміналу.
Хостинг = бот працює 24/7 сам, навіть коли комп'ютер вимкнено.

**Вартість:** Railway — приблизно $5/міс (хобі-план). Для бізнесу це копійки,
зате бот завжди онлайн і готовий до нагадувань (Етап 4).

---

## Частина 1 — GitHub (сховище коду)

### 1.1 Створи акаунт
Перейди на https://github.com → **Sign up** → зареєструйся.

### 1.2 Встанови GitHub Desktop
Завантаж https://desktop.github.com → встанови → увійди своїм акаунтом.
Це програма з кнопками, без командного рядка.

### 1.3 Створи репозиторій з папки бота
1. У GitHub Desktop: **File → Add Local Repository**
2. Обери папку `beauty_shine_bot`
3. З'явиться напис що це не репозиторій → натисни **"create a repository"**
4. Name: `beauty-shine-bot`
5. Постав галочку **Keep this code private** (важливо!)
6. Натисни **Create Repository**

### 1.4 Опублікуй
1. Натисни **Publish repository** (вгорі)
2. Переконайся що стоїть галочка **Keep this code private**
3. **Publish Repository**

> Файл `.env` НЕ потрапить на GitHub — він у списку `.gitignore`.
> Секретні ключі залишаються тільки в тебе. Це правильно.

---

## Частина 2 — Railway (хостинг)

### 2.1 Створи акаунт
Перейди на https://railway.app → **Login** → увійди через GitHub.

### 2.2 Створи проєкт
1. **New Project**
2. **Deploy from GitHub repo**
3. Дозволь Railway доступ до репозиторіїв → обери `beauty-shine-bot`
4. Railway почне деплой автоматично

### 2.3 Додай змінні середовища
1. Відкрий свій сервіс → вкладка **Variables**
2. Натисни **New Variable** і додай по черзі КОЖНУ змінну зі свого `.env`:

```
BOT_TOKEN
IRINA_CHAT_ID
IVAN_CHAT_ID
ANTHROPIC_API_KEY
AIRTABLE_TOKEN
AIRTABLE_BASE_ID
MASTER_PASSWORD
SALON_LAT
SALON_LNG
SALON_ADDRESS
SALON_PHONE
```

> Значення бери зі свого файлу `.env`. Назва зліва, значення справа.

### 2.4 Перевір команду запуску
1. Вкладка **Settings** → знайди **Start Command**
2. Якщо порожньо — впиши: `python bot.py`
3. Railway передеплоїть автоматично

---

## Частина 3 — Перевірка

### 3.1 Зупини локального бота
У терміналі на комп'ютері натисни **Ctrl+C**.

> ⚠️ ВАЖЛИВО: бот не може працювати одночасно на ПК і на Railway —
> буде конфлікт. Залишається тільки Railway.

### 3.2 Перевір логи
У Railway → вкладка **Deployments** → клікни на активний деплой →
маєш побачити `Beauty & Shine bot started ✅`

### 3.3 Перевір бота
Відкрий бота в Telegram → `/start` → пройди кілька розділів.
Тепер він працює без твого комп'ютера. 🎉

---

## Як оновлювати код далі

Коли я даю нову версію (Етап 4 тощо):
1. Заміни файли у папці `beauty_shine_bot`
2. Відкрий GitHub Desktop → побачиш список змінених файлів
3. Внизу зліва впиши коротко що змінив → **Commit to main**
4. Натисни **Push origin**
5. Railway САМ підхопить зміни і передеплоїть за 1-2 хвилини

Все. Жодного терміналу.
