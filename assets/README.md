# Картинки для бота

Бот вже використовує:
- `welcome_banner.jpg` — банер при /start
- `main_menu.jpg` — фото поверх головного меню клієнта
- `maintenance.jpg` — фото при тех-режимі

## Залишилось додати:

### 1. Фото Ірини (для розділу «Манікюр»)
- Назва файлу: **`master_irina.jpg`**
- Розмір: 800×800 пікселів (квадрат) або менше
- Формат: **JPG** (не PNG — швидше надсилається)
- Вага: до 200 КБ
- Як додати: просто перетягни файл у цю папку поверх `master_irina.jpg.placeholder` (плейсхолдер видали).

### 2. Фото Івана (для розділу «Педикюр»)
- Назва файлу: **`master_ivan.jpg`**
- Розмір: 800×800 пікселів (квадрат) або менше
- Формат: **JPG**
- Вага: до 200 КБ
- Як додати: так само, замість `master_ivan.jpg.placeholder`.

## Що буде якщо фото не додати?

Бот автоматично надсилатиме тільки текст-опис майстра. Помилки не буде —
просто без картинки. Як додаси — фото з'явиться без жодних змін у коді.
