# 🤖 Beauty & Shine — Telegram Бот

Покрокова інструкція налаштування з нуля.

---

## Крок 1 — Встановіть Python

1. Перейдіть на https://python.org/downloads
2. Завантажте Python 3.11 або новіший
3. Під час встановлення поставте галочку **"Add Python to PATH"**
4. Перевірте у терміналі:
   ```
   python --version
   ```
   Має вивести: `Python 3.11.x`

---

## Крок 2 — Створіть Telegram бота

1. Відкрийте Telegram → знайдіть **@BotFather**
2. Надішліть `/newbot`
3. Ім'я: `Beauty & Shine`
4. Username (латиниця, закінчується на bot): `beauty_shine_izmayil_bot`
5. Скопіюйте **токен** — виглядає так: `7123456789:AAHxxx...`

---

## Крок 3 — Дізнайтесь Chat ID майстрів

1. Ірина і Іван відкривають Telegram → знаходять **@userinfobot**
2. Надсилають `/start`
3. Бот відповідає з `Id: 123456789` — це Chat ID

---

## Крок 4 — Отримайте ключ Claude API

1. Перейдіть на https://console.anthropic.com
2. Зареєструйтесь або увійдіть
3. Зліва → **API Keys** → **Create Key**
4. Скопіюйте ключ (починається з `sk-ant-...`)

---

## Крок 5 — Налаштуйте Airtable

### 5.1 Зареєструйтесь
Перейдіть на https://airtable.com → безкоштовна реєстрація.

### 5.2 Створіть базу та таблиці
1. Натисніть **"Add a base"** → назвіть `Beauty & Shine`
2. Створіть три таблиці з такими стовпцями:

**Таблиця `Записи`:**
`Дата` · `Час` · `Імʼя` · `Username` · `Послуга` · `Деталі` · `Ціна` · `Тривалість` · `Нотатки ІІ`

**Таблиця `Заблоковані`** (коли майстер недоступний):
`Дата` · `Час` · `Причина`

**Таблиця `Дзвінки`** (заявки на дзвінок):
`Імʼя` · `Username` · `Телефон` · `Питання` · `ІІ-прапор` · `Дата заявки`

> Тип усіх полів — **"Single line text"**

### 5.3 Отримайте Base ID
З URL вашої бази скопіюйте Base ID:
`https://airtable.com/` **`appXXXXXXXXXXXXXX`** `/...`
Починається з `app`.

### 5.4 Створіть токен
1. Перейдіть на https://airtable.com/create/tokens
2. **Create token**
3. Name: `beauty-shine-bot`
4. Scopes: `data.records:read` + `data.records:write`
5. Access: **Add a base** → оберіть `Beauty & Shine`
6. **Create token** → скопіюйте токен одразу (показується лише раз!)

---

## Крок 6 — Налаштуйте файл .env

1. Скопіюйте `.env.example` → перейменуйте на `.env`
2. Заповніть:

```
BOT_TOKEN=7123456789:AAHxxx...
IRINA_CHAT_ID=123456789
IVAN_CHAT_ID=987654321
ANTHROPIC_API_KEY=sk-ant-...
AIRTABLE_TOKEN=patXXXXXXXXXXXXXX.XXXXXX...
AIRTABLE_BASE_ID=appXXXXXXXXXXXXXX
```

---

## Крок 7 — Запустіть бота

Відкрийте термінал у папці з проєктом:

```bash
# Встановіть залежності (один раз)
pip install -r requirements.txt

# Запустіть бота
python bot.py
```

Якщо все правильно — побачите: `Beauty & Shine bot started ✅`

---

## Крок 8 — Перевірте бота

1. Відкрийте Telegram → знайдіть свого бота
2. Натисніть `/start`
3. Перевірте всі 4 розділи

---

## Крок 9 — Постійний запуск (Railway)

Щоб бот працював 24/7:

1. Зареєструйтесь на https://railway.app
2. Підключіть GitHub і завантажте туди папку з ботом
3. В Railway → **"New Project"** → **"Deploy from GitHub"**
4. У **"Variables"** додайте всі змінні з `.env`
5. Railway автоматично запустить бота

---

## Управління розкладом

Відкрийте Airtable → таблиця **`Заблоковані`**:

| Дата | Час | Причина |
|------|-----|---------|
| 2025-06-15 | ВЕСЬ ДЕНЬ | відпустка |
| 2025-06-20 | 10:00 | особисте |

- `ВЕСЬ ДЕНЬ` — блокує весь день у календарі клієнта
- Конкретний час — блокує лише цей слот
